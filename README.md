# local-lens
A cool project for locating stores near you for things need; reducing your hassle for unhelpful web searches. 

#SETUP - BACKEND
```bash
    cd backend/
    python -m venv venv
    source venv/Scripts/activate (in bash)
    pip install -r requirements.txt
    python main.py
```
